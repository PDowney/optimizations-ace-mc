# Optimizations ACE MC

A lightweight WordPress optimization plugin with pre-configured performance enhancements for WooCommerce, WP Store Locator, and WordPress admin interfaces.

## Requirements

- WordPress 6.6+
- PHP 8.1+
- WooCommerce 5.0+
- WP Store Locator

## Features

- **WooCommerce**: Show empty categories, hide category product counts, user order count column
- **WP Store Locator**: Display store categories in info windows, disable REST API endpoint
- **WordPress Admin**: User registration date column with sorting

All features are individually configurable via **Settings > Optimizations ACE MC**.

## Installation

1. Upload the plugin to `/wp-content/plugins/optimizations-ace-mc`
2. Activate via the WordPress Plugins screen
3. Configure at **Settings > Optimizations ACE MC**
